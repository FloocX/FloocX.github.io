前言

​ 前段时间刚刚搭建起这个个人博客，hexo用的多了后，每次要进行操作时都要在hexo目录打开 git bash ，感觉比较麻烦，于是找到了hexon:

​ 但是使用过后发现写文章的时候没法插入图片，当然有其他师傅对这个插件进行了扩展，解决了无法插入图片等问题。

​ 因为我是用不惯这种运行服务然后在浏览器中操作的模式，所以自己用qt6写了个用来创建、管理文章的工具。

项目地址

这个项目因为是为了方便自己的，所以写的时候没有考虑那么多，而且最近很多事情要忙，功能只实现了大部分，个别功能没写，但不影响我自己的使用了。代码写的很像一坨了，有的地方实现的也比较别扭，有需要的师傅可以下载源码，根据自己的需求修改。
